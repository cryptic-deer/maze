# Maze

## Description

- Randomly generated maze on every page refresh
- Use WASD or Arrow keys to move the ball
- Created with the help of [matter.js](https://brm.io/matter-js/)

## Try it out

- [Play a game](https://cryptic-deer.github.io/maze/)

## Built by using

- Html
- CSS
- JavaScript
